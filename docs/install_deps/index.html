Install Dependecies

caution

plus only support development via Linux/OSX

If your're using Windows, this might but is not guaranteed to works. (use virtualmachines instead)

For most of our work, you should develop your application in Docker Containers.

If you don't want to connect to database or other services, (only write code in editor and wanna get syntax support/intellisence) try local development with syntax supports.

For gerenating docs, checkout Dependencies for writing docs.

Install dependecies via Docker#

  • Docker.
  • Docker Compose.

After installing docker and docker-compose, build your own docker image with:

docker-compose build

Local development with syntax support#

Python#

  • Install pyenv via pyenv-installer to have a virtual python version

    curl https://pyenv.run | bash

  • Install Python 3.7.7 via pyenv

    pyenv install 3.7.7

Poetry#

  • Intall Poetry for Package management tool for Python

    curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python3

  • Install dependecies by:

    poetry install

Using Poetry in VSCode#

Select the correct interpreter in your VSCode: vscode-python/issues/8372
